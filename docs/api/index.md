# API Reference

Full API documentation for all `secdashboards` modules. Each page documents the public classes, functions, and types exported by the module.

| Module | Description |
|--------|-------------|
| [`catalog`](catalog.md) | Data source catalog management |
| [`connectors`](connectors.md) | Athena, Security Lake, and CloudWatch Logs connectors |
| [`detections`](detections.md) | Detection rules framework and runner |
| [`graph`](graph.md) | Investigation graphs, timelines, Neptune persistence |
| [`notifications`](notifications.md) | Alert delivery via SNS and Slack |
| [`reports`](reports.md) | LaTeX/PDF report generation |
| [`ai`](ai.md) | Amazon Bedrock AI assistance |
| [`health`](health.md) | Data source health monitoring |
| [`adversary`](adversary.md) | Red team and adversary emulation |
| [`deploy`](deploy.md) | Lambda deployment utilities |
| [`cli`](cli.md) | Command-line interface |
